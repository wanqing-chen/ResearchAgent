## 1. 项目初始化阶段
在项目根目录（或者存放 Go 代码的文件夹，如你的 GoModule 目录）下执行：

```
go mod init <模块名>
```

## 2. 依赖管理阶段（按需执行）

```
go mod tidy
```

## 3. 编译与运行阶段（日常操作）

```
go run <文件名.go>
```

## 3. 开发go基本流程

原则：一个项目 = 一个 go.mod = 一次 go mod init。

日常流程：

1. 新建文件夹。

```
go mod init <名>（仅此一次）。
```

2. 写 .go 文件。

3. go mod tidy（增加新依赖时运行）。

4. go run <文件名>。
